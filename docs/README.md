# learn-next15-react19 Knowledge Base

このフォルダは Codex / Cursor から参照される一次知識ベースです。プロジェクト固有の前提や設計方針をここに集約し、生成 AI に渡す追加コンテキストとして活用します。

## ディレクトリ構成

- `architecture/` – アプリケーション構成や依存関係の概要
- `development/` – 開発セットアップ、ワークフロー、コーディング規約
- `knowledge/` – タスク完了時に追記するナレッジ更新用テンプレート
- `troubleshooting/` – よくある課題と対応策のメモ

## メンテナンスの原則

1. **常に一次情報を反映する** – 設計変更や依存パッケージの更新が発生したら、同一 PR 内でドキュメントを更新する
2. **リンクで連携する** – `.cursor/rules/*.mdc` から参照されるため、ファイル名の変更は PR で明示する
3. **フェーズ 5 で更新する** – Codex の `/knowledge` フェーズ終了時に差分を確認し、必要な情報を追加する
